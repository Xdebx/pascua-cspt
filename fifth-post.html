Computer Platforms • February 18th, 2023

Topic #5

Computers have been around for a long time. In the past, computers were only used by scientists and engineers to perform calculations and solve problems. However, in the last few decades, computers have become more common in everyday life. Computers are now used for entertainment purposes such as playing games or watching videos on YouTube. They are also used for work purposes such as banking or paying bills online. Computing platforms are the hardware and software that make up a computer system. They are the foundation on which all other components of a computer system depend. A computing platform is not just one thing, but rather it is an evolving combination of many different elements that work together to provide an environment for running programs and performing tasks.

  • cloud computing
  • virtualization

Cloud computing is a new term that describes the delivery of computer services and software as a service. This means that users can access these services from anywhere, on any device, at any time. Cloud computing has many advantages over traditional methods of storing data and information because it allows for easy accessibility to all data without having to store it on local devices. The main advantage of cloud computing is its cost-effectiveness because it uses less energy than traditional methods of storing data. Another advantage is that users can use their devices while they are working or playing online without having to worry about losing their progress or settings since everything is stored in the cloud. There are many different types of clouds such as public, private, community and hybrid clouds. Public clouds are shared by multiple organizations while private clouds are only used by one organization but they may be shared with other organizations if they choose to do so. Public cloud resources can be accessed from anywhere, at any time, and from any device. Private cloud resources can only be accessed from the company that owns the server/storage unit within the cloud. Cloud storage is a great way to store data because it is secure and backed up in case of an emergency or disaster.

Community clouds allow multiple organizations to share resources within the cloud but not necessarily with each other which means there would be no security issues when using this type of cloud due to lack of between communication each other’s systems/devices/data storage units. Hybrid Clouds combine both public and private resources. There are many different cloud services that organizations can use such as Office 365, Google Drive, Dropbox, AWS etc. These cloud services allow users to store files online in the cloud and access them from any device at any time. There are also many different file sharing apps that allow users to share files with other people or devices such as WhatsApp, Skype, Facebook Messenger etc. Cloud computing has made working from home or remotely possible because it allows for easy accessibility of information while still being able to work on your own schedule without having to worry about losing your progress or settings since everything is stored in the cloud.

Virtualization has been around since the 1970s on mainframes and minicomputers, but it wasn’t until the mid-1990s that virtualization became popular on personal computers. The first type of virtualization was called paravirtualization because it used a layer of software emulation to mimic the behavior of a real computer. This allowed programs written for one type of computer to run on another type without any changes being required. Paravirtualization is still in use today, but it has been supplemented by full virtualization, which allows complete isolation from hardware resources and direct access to memory and peripheral devices. Virtual machines can be created from physical computers or from other virtual machines.

Virtualization is the process of making a virtual machine. A virtual machine is an independent computer that runs on top of another operating system, and it has its own storage area. Virtual machines can be created from physical computers or from other virtual machines. The main purpose of virtualization is to create multiple identical computers that are easier to manage and maintain. Virtualization can be used for many different purposes, but one of the most common uses for it is in server management. Server consolidation allows multiple servers to run on a single physical server, which reduces hardware costs and increases security because fewer people have access to the server at any given time. Another common use for virtualization is desktop consolidation, which involves running multiple individual workstations on a single desktop computer instead of having each person use their own workstation. This allows employees with specialized tasks (such as graphic designers) to work together more easily instead of having them sit behind separate terminals while everyone else completes their tasks using a shared document library or email account.
