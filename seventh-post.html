Storage Platforms • March 3rd, 2023

Topic #7

Storage platforms There are many different types of storage platforms. Some of the most common ones include hard drives, flash drives, and solid state drives. These are all used to store data on a computer or other electronic device. Hard drives are the most common type of storage platform because they have been around for a long time and can hold large amounts of data. Flash drives were made to be more convenient than having to use CDs or DVDs every time you wanted to save something, but they don’t last as long as hard drives do so you will need to replace them more often. Solid state drives (SSDs) were made in order to compete with hard disks because they both store data but SSDs have faster speeds and don’t wear out as fast as HDDs do.

There is one main difference between these three types of storage platforms: how fast the platform is when reading/writing files from it or saving them onto it (IOPS). An IOPS stands for Input/Output Operations Per Second and measures how quickly a drive can process information that needs to be saved onto it or sent from it into memory while being read from memory into the system’s RAM (Random Access Memory). An SSD has the highest IOPS, a hard drive has the second highest IOPS, and a flash drive has the lowest IOPS.

Hard Disk Drives (HDDs): HDDs are the most common type of storage platform. They are relatively inexpensive and offer large amounts of storage capacity. They work by using a spinning disk to store data and read/write heads to access that data.

‘Solid State Drives (SSDs): SSDs are faster than HDDs because they use flash memory to store data, which doesn't require any moving parts. They are more expensive than HDDs but are becoming increasingly popular due to their speed and reliability.

Network Attached Storage (NAS): NAS is a type of storage platform that is connected to a network, making it accessible to multiple users. It can be used to store files and backups, and is often used in small business or home environments.

Storage Area Networks (SANs): SANs are designed to provide high-speed access to storage for large-scale enterprise environments. They use dedicated network infrastructure to connect servers and storage devices.

Cloud Storage: Cloud storage involves storing data on remote servers that can be accessed over the internet. This allows for easy sharing and access to data from anywhere with an internet connection.

Tape Storage: Tape storage uses magnetic tape to store data. It is slower than other types of storage platforms but is often used for long-term data retention and backup purposes.

Optical Storage: Optical storage uses lasers to read and write data to discs such as CDs, DVDs, and Blu-ray discs. It is commonly used for data backup and distribution purposes.
